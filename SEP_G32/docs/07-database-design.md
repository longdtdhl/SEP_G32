# 07-database-design.md

# 1. Purpose

This document defines the physical database structure of the Online Psychological Counseling Booking System (OPCBS).

The database design serves as the primary reference for:

* SQL Server database implementation
* Entity Framework Core entities
* Repository layer development
* API implementation
* Business rule enforcement
* Data migration strategy

---

# 2. Database Standards

## 2.1 Database Engine

```text
SQL Server 2022
```

---

## 2.2 Primary Key Standard

All tables use:

```text
Id UNIQUEIDENTIFIER
```

Generated by:

```sql
NEWSEQUENTIALID()
```

---

## 2.3 Audit Fields

All aggregate tables must contain:

| Column    | Type                  |
| --------- | --------------------- |
| CreatedAt | DATETIME2             |
| UpdatedAt | DATETIME2             |
| CreatedBy | UNIQUEIDENTIFIER NULL |
| UpdatedBy | UNIQUEIDENTIFIER NULL |

---

## 2.4 Soft Delete

Applicable entities:

* Users
* DoctorProfiles
* Packages
* BlogPosts

Columns:

| Column    | Type |
| --------- | ---- |
| IsDeleted | BIT  |

---
